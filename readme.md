# GasCert Pro - Enterprise Gas Certificate Management System

This compliance system transforms custom inputs, smart autocompletes, and dynamic automation rule structures into printable A4 Landscape safety certificates.

## Feature Overview

1. **Dashboard Interface**: Unified analytical pane tracking completed, draft, expiring, and active certificate volumes offline.
2. **Dynamic Rules Automation**: Custom user rules populate columns instantaneously based on the target appliance (e.g., `IF Appliance is Cooker, THEN Flue = N/A`).
3. **Reactive Autocomplete**: Real-time billing addresses, previous installation properties, and engineer licenses index instantly as you type.
4. **Drawing Signature Engine**: Responsive mouse-to-touch canvas for signing off certificates.
5. **A4 Landscape Print Output**: When compiled, the styles align perfectly with standard regulatory formats, bypassing manual entry.

## Local Execution Instructions

1. Put `index.html`, `style.css`, and `script.js` inside the same directory.
2. Open `index.html` inside Google Chrome or Microsoft Edge.
3. Click **"Populate Mock System Data"** on the Dashboard to load default demo variables instantly.
4. Go to **"New Certificate"** to construct certificates in real-time.
5. Click **"Print / Generate PDF"** on the print screen to view the pixel-perfect landscape layout.
